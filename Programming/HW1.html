Blackbox Test System

Instruction

Write a program that reads two integers and outputs their sum.

  • Input: two integers
  • Range: -2 × 10⁹ to 2 × 10⁹
  • Output: a single integer

Blackbox testing is applied. Only input/output behavior matters.

Case 0 checks whether the program can run. Cases 1–10 are hidden test cases.
